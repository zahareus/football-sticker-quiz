## 13. Рекомендації (пріоритезовані)

### ВИСОКИЙ пріоритет

1. **Перекроулити Bodø/Glimt та інші збагачені сторінки**
   Топ сторінка не кроулилась 32 дні -- Google бачить стару версію без Wikipedia-контенту. Подати на повторний кроул через GSC або оновити sitemap lastmod.

2. **Оптимізація snippet для "sticker [club]" запитів**
   113 показів з 0% CTR для Bodø/Glimt. Потрібно A/B тест title: замість "Bodø/Glimt Stickers" спробувати "Bodø/Glimt Sticker Collection -- 5 Football Stickers | StickerHunt" з кращим описом у meta description.

3. **Контентне збагачення для довгого хвоста**
   560/675 клубів мають Wikipedia-контент. Решта 115:
   - **4 клуби** мають Wikipedia URL, але скрипт не зміг витягти дані (AC Pavia, FK Zemun, OFK Belgrade, Vitesse Arnhem) -- перезапустити fetch-wiki-data.js для них
   - **111 клубів** не мають Wikipedia взагалі (аматорські/нижчолігові). З них 83 мають сайт клубу, 16 тільки Instagram, 11 тільки Facebook
   Для 4 клубів -- фікс скрипту. Для 111 -- альтернативне збагачення (Transfermarkt, сайти федерацій) або AI-generated описи.

### СЕРЕДНІЙ пріоритет

4. **Мультимовні meta descriptions для топ-ринків**
   Нідерланди, Німеччина, Норвегія -- додати hreflang або мультимовні мета для ключових запитів (nld: "voetbalstickers", deu: "Fußball Aufkleber").

5. **Розширити city pages**
   Міські сторінки проіндексовані з Breadcrumbs -- але ще не приносять трафік. Збагатити контентом (факти про місто, футбольна культура).

6. **Робота з Image Search**
   Попередній звіт показав потенціал -- оптимізувати alt text та file naming для sticker images.

### НИЗЬКИЙ пріоритет

7. Моніторити спад трафіку в кінці березня -- якщо продовжиться в квітні, потрібна діагностика.
8. Розглянути llms.txt та розблокування AI-ботів.
